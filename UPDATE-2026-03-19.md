2. 统一一个 skill 内的两条核心工作流：
   - Top10 小时级市场巡检
   - Binance USDS-M 合约 `report / alert / heartbeat / json / report+json`
3. 新增合约分析端点说明：
   - `references/usds-futures-endpoints.md`
4. 更新 `SKILL.md` 与 `agents/openai.yaml`，让能力描述与调用方式更加完整
5. 更新 `README.md`，补充独立 skill 发布所需的安装、使用与 cron 示例
6. 更新 `references/openclaw-setup.md`，补充内置合约 heartbeat 的接入方式

## 功能概览

发布后的 `binance-market-watch` 支持：

- 每小时扫描 Binance 高成交量 `Top 10` USDT 标的
- 识别 `long_signal`、`short_signal`、`margin_stress`
- 输出 Telegram 友好的监控摘要
- 输出结构化 `json`
- 运行 Binance USDS-M 合约 `heartbeat`
- 生成合约 `report` 与 `alert`
- 提供 `report+json` 复合输出
